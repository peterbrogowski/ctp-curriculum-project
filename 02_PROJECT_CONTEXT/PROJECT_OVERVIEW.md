# Project Overview

## Project name
CTP Curriculum Project

## Purpose
Develop a comprehensive, evidence-based curriculum aligned to the **ABTC Certified Transplant Preservationist (CTP)** certification exam blueprint.

## Audience
- 6 Organ Machine Perfusion Specialists
- 4 staff expected to be exam-eligible within a year
- 2 early trainees participating in curriculum development and learning
