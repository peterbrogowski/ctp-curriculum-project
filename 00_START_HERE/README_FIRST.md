# CTP Curriculum Project — Start Here

## What this project is
This repository is a continuity export for the **CTP Curriculum Project**, a professional training curriculum being developed for the **ABTC Certified Transplant Preservationist (CTP)** certification exam.

## Core project goals
1. Extract and structure the ABTC CTP exam blueprint
2. Analyze topic weighting and priorities
3. Design the curriculum architecture
4. Develop detailed teaching modules
5. Create practice questions
6. Compile a final study manual
