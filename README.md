# CTP Curriculum Project

## Overview
This repository contains the full continuity export and development structure for the ABTC Certified Transplant Preservationist (CTP) curriculum project.

## Purpose
Develop a professional, evidence-based training curriculum aligned with the ABTC CTP certification exam.

## Structure
- `00_START_HERE` → onboarding
- `01_SYSTEM` → rules + protocols
- `02_PROJECT_CONTEXT` → scope + constraints
- `03_ACTIVE_PROJECTS` → status + next steps
- `04_WORKFLOWS` → execution patterns
- `05_DATA_MODELS` → structure + schema
- `06_ARTIFACTS` → outputs + templates
- `07_HANDOFF` → AI startup prompt
- `08_LOGS` → session tracking
- `09_JSON_STATE` → machine-readable state
- `10_IMPORTS_RAW` → source files
